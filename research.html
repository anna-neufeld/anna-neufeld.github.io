Software

  • R package splinetree implements regression trees and random forests for longitudinal data using a modified version of a spline projection method first proposed by Yu and Lambert (1999). The modified method allows users to build trees while paying attention to either a full trajectory or only the shape of a trajectory (ignoring its level). The R package is available on CRAN ./
